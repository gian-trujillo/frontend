# Descripcion

-El sitio Skylens Adventures es una aplicacion fullstack para un fotografo (Gian Trujillo) en donde muestra su trabajo, habla de su experiencia y ofrece servicios fotograficos.
-Utilizando Javascript, principalmente React, se creo el frontend responsivo con menus desplegables, videos embebidos desde youtube y un panel de administracion protegido con autenticacion JWT.
-Integracion con Cloudinary permite el manejo de imagenes y videos cortos en el panel de administracion sin modificar directamente el codigo frontend.
-Tambien se integro la api Resend para el envio de formularios al correo electronico del administrador desde la seccion/pagina de contacto.
-El backend se desarollo con Node.js y Express para processar la informacion enviada del frontend y se utilizo MongoDB para almacenar esta informacion y la recibida por Cloudinary.
-Este sitio tiene como proposito el presentar informacion a los visitantes de manera clara y elegante, mientras le da accesso al fotografo para editar y editar contenido y precios.
